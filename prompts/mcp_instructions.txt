- If `read_omd_file` is called on a directory, it will return an error and the directory listing.
- `read_odt_placeholders` takes the absolute path of an ODT template and extracts all unique placeholders of the form `{{placeholder}}` inside it.
- `modify_odt_file` copies a template ODT file and substitutes specific strings/placeholders (keys of replacements dict) with new values, keeping the XML structure valid via automatic escaping. Use this to construct new documents based on template documents!
